# Sungmin's Blog

GitHub Pages와 Jekyll Chirpy 테마로 운영하는 개인 블로그입니다.

- URL: https://mochabbang.github.io
- 카테고리: 공부, 회고, 생각 정리
- 배포: GitHub Actions → GitHub Pages

## 글 작성

`_posts/YYYY-MM-DD-title.md` 형식으로 Markdown 파일을 추가합니다.

```yaml
---
title: 글 제목
date: YYYY-MM-DD HH:MM:SS +0900
categories: [공부]
tags: [jekyll, github-pages]
---
```
